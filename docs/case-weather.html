Weather Forecast App

This interactive web app connects to the Open-Meteo API to display real-time weather information and a 7-day temperature forecast for any selected city. Users can choose a location and instantly view daily maximum and minimum temperatures — showcasing how API integration and front-end development can provide dynamic, data-driven insights.
