Exploring geospatial data with FastAPI, Docker, and OpenStreetMap.

Emalytics POI Finder

Emalytics POI Finder is a full-stack web application developed to explore and visualize nearby Points of Interest (POIs) such as schools, pharmacies, and restaurants using OpenStreetMap data.

Built with FastAPI and Leaflet.js, and fully containerized with Docker, this project demonstrates how geospatial APIs can be efficiently deployed and scaled using modern cloud infrastructure like Render.com.
